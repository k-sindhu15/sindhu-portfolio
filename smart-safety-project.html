Project Description

The Smart Safety System for Mining Workers is an IoT-based solution designed to enhance worker safety in hazardous mining environments. The system continuously monitors environmental conditions and worker health parameters and provides real-time alerts to prevent accidents.

Key Features

  • Gas leakage detection (toxic gases)
  • Temperature and humidity monitoring
  • Emergency alert system
  • Real-time data monitoring
  • Buzzer alert in critical conditions

Technologies Used

  • Arduino / Microcontroller
  • IoT Sensors (Gas, Temperature, Humidity)
  • Embedded C / Arduino IDE
  • Wireless communication modules

Outcome

The project improves worker safety by providing early warnings and real-time monitoring, significantly reducing the risk of accidents in mining environments.
